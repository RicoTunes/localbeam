How to Use

  1. Make sure your phone and laptop are connected to the same Wi-Fi network
  2. Scan the QR code with your phone's camera or any QR scanner app
  3. Open the link in your phone's browser
  4. Browse and select files to download to your phone
  5. Use the upload button to send files from phone to laptop
  6. For fastest transfers, keep the app open on both devices
Pro Tip: Use Chrome or Firefox on Android for best performance.

About Wireless File Transfer

This app enables blazing fast file transfers between your laptop and Android device without USB cables.

Features:

  • QR code scanning for easy connection
  • Fast file transfers over local Wi-Fi
  • Real-time file browsing
  • Clipboard sharing between devices
  • No installation required on phone
  • Secure local network transfer

Built with Python, Flask, and modern web technologies.
